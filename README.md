# Movie-Sales-Dashboard


1. Project Title / Headline
An interactive, visually rich Power BI dashboard built to understand what makes a movie successful by comparing top films, directors, genres, and performance metrics.

 2. Short Description / Purpose

The Movie Analytics Dashboard provides a clear and interactive way to explore movie data, focusing on revenue, ratings, votes, runtime, actors, and directors.
It helps users quickly identify what drives movie success and how different films compare across key performance indicators.
